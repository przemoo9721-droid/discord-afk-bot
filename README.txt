
# Discord AFK Bot

Bot automatycznie przenosi użytkownika na kanał AFK, gdy się wyciszy.

## Instalacja

1. Zainstaluj Node.js:
https://nodejs.org

2. Otwórz terminal w folderze bota

3. Wpisz:
npm install

4. Otwórz plik index.js

5. Podmień:
PASTE_BOT_TOKEN_HERE
na token bota

6. Podmień:
PASTE_AFK_CHANNEL_ID_HERE
na ID kanału AFK

## Jak zdobyć token

Discord Developer Portal:
https://discord.com/developers/applications

- New Application
- Bot
- Reset Token
- Skopiuj token

## Jak zdobyć ID kanału

Discord:
Ustawienia → Zaawansowane → Tryb deweloperski

Prawy klik kanału → Kopiuj ID

## Uruchomienie

W terminalu:
npm start

## Wymagane uprawnienia bota

- Move Members
- View Channels
- Connect

Bot musi mieć rolę wyżej niż użytkownicy.
